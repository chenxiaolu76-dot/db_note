<span class="key-point">Lecture 0 不属于期末考试知识点主体，但它决定了课程结构、教材范围、作业与实验安排。</span>

## 1 Overview

Lecture 0 不是数据库知识点本身，而是整门课的导航页。它主要回答三件事：

- 这门课学什么
- 用哪本教材
- 课程、实验、作业、考试如何安排
