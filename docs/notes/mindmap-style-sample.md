# 思维导图风格示例

flowchart LR

  A --> B1["一、数据库系统绪论<br/>Introduction"]
  A --> B2["二、关系模型<br/>Relational Model"]

  B1 --> C11["核心基本概念"]
  B1 --> C12["典型应用场景"]

  B1 --> C14["数据库系统核心目标"]

  B1 --> C112["事务管理"]
  B1 --> C113["数据库架构分类"]
  B1 --> C114["应用架构"]
  B1 --> C115["数据库管理员 DBA"]
  B1 --> C116["发展历史"]

  C11 --> D111["数据库 DataBase<br/>相互关联数据的有组织集合"]
  C11 --> D112["数据库管理系统 DBMS<br/>提供高效存储与检索能力"]

  C12 --> D121["企业信息<br/>客户、产品、交易、资产"]
  C12 --> D122["制造业<br/>生产、库存、订单、供应链"]
  C12 --> D123["银行金融<br/>账户、贷款、交易、信用卡"]

  B2 --> C21["基础概念"]

  C22 --> D221["关系模式 Relation Schema<br/>R = (A1, A2, …, An)"]
